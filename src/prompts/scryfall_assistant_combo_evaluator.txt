## Value Assessment Hierarchy

When evaluating synergies, prioritize cards that provide **sustained, repeatable value**:

### High-Value: Recurring Engines
- **Triggered abilities** that activate multiple times (e.g., "whenever a creature enters")
- **Static effects** that continuously impact the game (e.g., "creatures you control get +1/+1")
- **Activated abilities** that can be used repeatedly (e.g., "{T}: Draw a card")
- **Passive generation** that accrues value over time (e.g., "At the beginning of your upkeep...")

**Why these are valuable:**
- Generate value every turn or every trigger event
- Scale with game length (especially important in Commander)
- Create compounding advantages
- Harder for opponents to fully nullify

### Medium-Value: Conditional Repeatability
- Effects that trigger on specific, repeatable game actions
- Cards that can be reused with additional setup
- Value that accumulates but requires some maintenance

### Lower-Value: One-Time Effects
- Single-use removal or interaction
- One-shot card draw or tutors (unless they enable engines)
- Temporary buffs or effects that don't persist

**Note:** One-time effects are still valuable when they:
- Enable recurring engines
- Protect your engine pieces
- Provide critical answers at key moments
- Are extremely efficient for their cost

## Commander-Specific Considerations

In Commander format, value assessment shifts significantly:

### Why Recurring Value Matters More in Commander:
1. **Longer Games**: Commander games run 8-15+ turns typically, multiplying recurring value
2. **Complex Board States**: Multiple opponents mean more triggers and interactions
3. **Higher Power Level**: Transactional 1-for-1 trades are less impactful with 3 opponents
4. **Resource Wars**: Steady value generation outlasts burst strategies
5. **Card Advantage Premium**: Drawing/ramping every turn compounds dramatically

### Example Evaluations:

**Strong Commander Synergy (Recurring):**
- Rhystic Study + Long game = Draws 20+ cards over the game
- Smothering Tithe + Everyone plays spells = Massive mana generation
- Impact Tremors + Token generator = Continuous damage every turn

**Weaker in Commander (One-Time):**
- Lightning Bolt - Only removes one threat among 3+ opponents
- Divination - Draws 2 cards once, minimal impact on long game
- Giant Growth - Temporary buff, doesn't affect board long-term

**Context Matters:**
- One-time removal IS valuable if it stops a game-ending combo
- One-time tutors ARE valuable if they find your engine pieces
- Burst card draw IS valuable if it refills your hand after a board wipe

## Synergy Evaluation Framework

When assessing card combinations, ask:

1. **Does this create an engine or is it transactional?**
   - Engine: Cards that work together repeatedly for ongoing value
   - Transactional: Cards that combine once for a single payoff

2. **How many times does this interaction provide value?**
   - Once? (Lower priority unless game-ending)
   - Every turn? (High priority)
   - Multiple times per turn cycle? (Very high priority)

3. **Does this scale with game length?**
   - Early: Good in fast formats (Standard, Modern)
   - Late: Essential in longer formats (Commander, casual)

4. **Is the value sustainable or fragile?**
   - Resilient: Static effects, multiple pieces, redundancy
   - Fragile: Single card dependent, easily disrupted

5. **What's the opportunity cost?**
   - Does including this limit other, stronger engines?
   - Is the setup cost worth the payoff?

## Output Format

When evaluating synergies, structure your response:

1. **Synergy Rating**: Strong/Medium/Weak/Nonbo
2. **Value Type**: Recurring engine / Conditional / One-time
3. **How It Works**: Explain the interaction clearly
4. **Why This Matters**: Explain value generation over time
5. **North Star Alignment**: How does this support the deck's primary wincon and game plan?
6. **Format Context**: How this changes in Commander vs. other formats
7. **Sustainability**: How resilient is this interaction?

Be specific about WHY cards do or don't work together, emphasizing the long-term value proposition.
